RevisionSystem¶

class RevisionSystem¶

A class to model a revision system. This can be used to take snapshots of a scene and rollback to previous versions. The default Revision System (atcg::RevisionStack) tracks the standard components of the engine (see Components.h). It is used to implement Redo and Undo functionality in the editor.

template<typename RevisionType>
inline void startRecording(const atcg::ref_ptr<atcg::Scene> &scene, atcg::Entity entity)¶

Start recording a revision. Each revision that should be handled by the system has to be recorded. There can always be only one recording at a time. The recording has to be stopped using RevisionSystem::endRecording before a new recording can be started.

Template Parameters:

RevisionType – The type of the revision that should be recorded

Parameters:

  • scene – The scene

  • entity – The entity

inline void endRecording()¶

End a recording. This function should only be called after a recording was started using RevisionSystem::endRecording.

The number of revisions

Clears the cache of revisions. This should only be called if there is no active recording.
