template<typename T>
inline T atcg::Color::color_matching_x(const T lambda)¶

The color matching function x_bar. Uses the analytical approximation (https://en.wikipedia.org/wiki/CIE_1931_color_space#Color_matching_functions). No additional normalization is done. For converting spectral data to RGB via integration, normalize by the integral over Y (atcg::Constants::Y_integral())

Template Parameters:

T – The type

Parameters:

lambda – The wavelength in nm

Returns:

x_bar(lambda)

template<typename T>
inline T atcg::Color::D65(const T lambda)¶

Evalute the D65 standard illuminant function. Values have to be between 300nm and 780nm. The spectral data is normalized s.t. the discrete version of D65(560) = 100. However, due to interpolation there might be deviations from this value.

Template Parameters:

T – The type

Parameters:

lambda – The wavelength in nm

Returns:

The spectral radiance
