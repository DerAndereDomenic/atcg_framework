Layer¶

class Layer¶

The layer system is used to handle events and draw calls in a specific order In most of our cases we only have 1 layer -> the application layer that is created by the user.

Subclassed by ImGuiLayer

Public Functions

inline Layer(const std::string &name)¶

Construct a new Layer object.

Parameters:

name – The name of the layer

virtual ~Layer() = default¶

Destroy the Layer object.

inline virtual void onAttach()¶

This function gets called when the layer is created. Implemented by user.

inline virtual void onDetach()¶

This function gets called when the layer is destroyed Implemented by the user.

inline virtual void onUpdate(float delta_time)¶

This function gets called every frame. It is used for physics updates and rendering.

Parameters:

delta_time – The time since the last frame

inline virtual void onImGuiRender()¶

This function should only handle imgui code.

inline virtual void onEvent(Event *event)¶

The event handle callback.

Parameters:

event – The event to handle

Get the Name object.
