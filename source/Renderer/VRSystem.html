void init(const EventCallbackFn &callback)¶

Initialize the VR renderer.

Parameters:

callback – The callback function

void initControllerMeshes(const atcg::ref_ptr<atcg::Scene> &scene)¶

Initialize controller renderings. This has to be called by the client. Adds controller entities to the scene.

These entities get updated automatically based on the current pose of the tracker every frame.

Parameters:

scene – The scene

void onUpdate(const float delta_time)¶

Gets called every frame and handles VR updates.

Parameters:

delta_time – The delta frame time

void doTracking()¶

Do the tracking. This is a blocking call and should be called at the beginning of the rendering thread. This is done by the application, not the client.

void emitEvents()¶

This function pulls events from the headset and forwards them through the framework.
